File & Directory Navigation BASIC

ফাইল ও ডিরেক্টরি নেভিগেশন
Viewing Files & Folders
$ls
List files and folders in current directory.
বর্তমান ডিরেক্টরির ফাইল ও ফোল্ডার দেখায়।
$ls -la
Detailed list including hidden files (starting with dot).
লুকানো ফাইলসহ বিস্তারিত তালিকা দেখায়।
$ls -lh
Human-readable file sizes (KB, MB, GB).
ফাইলের সাইজ সহজে বোধগম্য ফরম্যাটে দেখায়।
ℹ️
Use ls -la to see hidden config files — attackers love them! / পেন্টেস্টাররা সবসময় ls -la দিয়ে লুকানো ফাইল চেক করে!
Moving Around
$pwd
Print current working directory — "Where am I?"
আমি এখন কোথায় আছি তা দেখায়।
$cd /etc
Change directory to /etc (config files live here).
/etc ডিরেক্টরিতে যায় — এখানে কনফিগ ফাইল থাকে।
$cd ~
Go to home directory — your personal safe space.
হোম ডিরেক্টরিতে ফিরে যায়।
$cd ..
Go up one directory level.
এক লেভেল উপরে যায়।
$cd -
Go back to previous directory — like a browser "back" button.
আগের ডিরেক্টরিতে ফিরে যায়।
Creating & Deleting
$mkdir test
Create a new directory named "test".
"test" নামে নতুন ফোল্ডার তৈরি করে।
$mkdir -p a/b/c
Create nested directories in one shot.
একসাথে নেস্টেড ফোল্ডার তৈরি করে।
$touch file.txt
Create an empty file or update its timestamp.
খালি ফাইল তৈরি করে বা টাইমস্ট্যাম্প আপডেট করে।
$rm -rf folder/
Force delete folder and all contents. NO UNDO!
ফোল্ডার ও সব কন্টেন্ট মুছে দেয়। কোনো আনডু নেই!
⚠️
rm -rf / deletes EVERYTHING. Never run this! / rm -rf / সবকিছু মুছে দেয়। কখনো এটা চালাবেন না!
Reading Files
$cat file.txt
Display entire file content at once.
পুরো ফাইলের কন্টেন্ট একসাথে দেখায়।
$head -n 20 file.txt
Show first 20 lines of a file.
ফাইলের প্রথম ২০ লাইন দেখায়।
$tail -f logs.txt
Live view of file as it updates — great for log monitoring.
ফাইল আপডেট হওয়ার সাথে সাথে লাইভ দেখায় — লগ মনিটরিংয়ে দুর্দান্ত।
$less file.txt
Scroll through file (press Q to quit).
ফাইল স্ক্রোল করে দেখায় (Q চাপলে বের হয়)।
💡
tail -f /var/log/auth.log — watch login attempts in real time! / লাইভ লগইন প্রচেষ্টা দেখার জন্য দুর্দান্ত!
Searching & Finding
$find / -name "*.conf" 2>/dev/null
Find all .conf files. Ignore permission errors.
সব .conf ফাইল খোঁজে। এরর মেসেজ উপেক্ষা করে।
$find . -type f -size +10M
Find files larger than 10MB.
১০ মেগাবাইটের বেশি ফাইল খোঁজে।
$grep -r "password" /etc/
Recursively search for "password" in /etc/ — pentesters' gold mine.
/etc/-এ রিকার্সিভলি "password" শব্দ খোঁজে — পেন্টেস্টারের সোনার খনি।
$grep -i "user" file.txt
Case-insensitive search — finds "User", "USER", "user".
ছোট/বড় হাতের পার্থক্য ছাড়া খোঁজে।
Copy, Move & Archive
$cp -r dir1 dir2
Copy an entire folder recursively.
পুরো ফোল্ডার রিকার্সিভলি কপি করে।
$mv old.txt new.txt
Rename a file (or move it to another location).
ফাইলের নাম পরিবর্তন করে বা স্থানান্তর করে।
$tar -czf files.tar.gz folder/
Compress a folder to .tar.gz archive.
ফোল্ডার কম্প্রেস করে .tar.gz ফাইল তৈরি করে।
$tar -xzf files.tar.gz
Extract .tar.gz archive.
.tar.gz আর্কাইভ এক্সট্র্যাক্ট করে।

Permissions & Ownership INTERMEDIATE

অনুমতি ও মালিকানা
chmod — Change Permissions
$chmod +x script.sh
Make a script executable. Step 1 before running any script.
স্ক্রিপ্ট এক্সিকিউটেবল করে। যেকোনো স্ক্রিপ্ট চালানোর আগে প্রথম ধাপ।
$chmod 755 file.sh
Owner=read+write+execute, Group=read+execute, Others=read+execute.
মালিক = পড়া+লেখা+চালানো, গ্রুপ = পড়া+চালানো, অন্যরা = পড়া+চালানো।
$chmod -R 755 /path/
Apply permissions recursively to entire directory.
পুরো ডিরেক্টরিতে রিকার্সিভলি অনুমতি প্রয়োগ করে।
💡
Remember: chmod 777 = everyone can do everything = a security nightmare! / মনে রাখুন: chmod 777 = সবাই সব করতে পারে = নিরাপত্তার দুঃস্বপ্ন!
SUID / SGID — Privilege Escalation Gold
$find / -perm -4000 -type f 2>/dev/null
Find ALL SUID binaries on the system — runs as the file owner (often root!).
সিস্টেমে সমস্ত SUID বাইনারি খোঁজে — ফাইল মালিকের (প্রায়ই root!) হিসেবে চলে।
$find / -perm -2000 -type f 2>/dev/null
Find SGID binaries — runs with group permissions of the file.
SGID বাইনারি খোঁজে — গ্রুপের অনুমতিতে চলে।
$chmod 4755 file
Set SUID bit — file runs as owner (dangerous if misconfigured).
SUID বিট সেট করে — ফাইল মালিকের হিসেবে চলে (ভুল কনফিগ হলে বিপজ্জনক)।
🚨
SUID on a shell binary = instant root shell for attackers! Check GTFOBins! / শেল বাইনারিতে SUID থাকলে = আক্রমণকারীর জন্য তাৎক্ষণিক root শেল!
chown & Special Bits
$chown user:group file.txt
Change file owner and group.
ফাইলের মালিক ও গ্রুপ পরিবর্তন করে।
$chown -R user:group /dir
Change owner recursively for entire directory.
পুরো ডিরেক্টরিতে রিকার্সিভলি মালিক পরিবর্তন করে।
$chmod +t /tmp
Set sticky bit — only file owner can delete files in that directory.
স্টিকি বিট সেট করে — শুধু ফাইল মালিক মুছতে পারে।
$umask
Show default permission mask for new files.
নতুন ফাইলের ডিফল্ট অনুমতি মাস্ক দেখায়।

Process & Network Monitoring INTERMEDIATE

প্রসেস ও নেটওয়ার্ক মনিটরিং
Process Management
$ps aux
List ALL running processes — every single one.
সমস্ত চলমান প্রসেস তালিকাভুক্ত করে।
$top
Interactive process viewer — like Task Manager for Linux.
ইন্টারেক্টিভ প্রসেস ভিউয়ার — লিনাক্সের Task Manager।
$kill -9 1234
Force kill process by PID. No mercy, no confirmation.
PID দিয়ে প্রসেস জোর করে মেরে ফেলে। কোনো দয়া নেই!
$pgrep nginx
Find PID of a process by name.
নাম দিয়ে প্রসেসের PID খোঁজে।
Network Commands
$ip addr
Show all network interfaces and their IP addresses.
সব নেটওয়ার্ক ইন্টারফেস ও IP দেখায়।
$ss -tuln
Show open ports. What services are listening?
খোলা পোর্ট দেখায়। কোন সার্ভিস শুনছে?
$lsof -i -P -n
List open network sockets with process info.
প্রসেস তথ্যসহ খোলা নেটওয়ার্ক সকেট দেখায়।
$tcpdump -i eth0
Capture network packets on interface. Needs root!
ইন্টারফেসে প্যাকেট ক্যাপচার করে। root দরকার!
ℹ️
ss -tuln is the modern replacement for netstat. / ss -tuln হলো netstat-এর আধুনিক বিকল্প।

Bash Scripts & Productivity INTERMEDIATE

ব্যাশ স্ক্রিপ্ট ও প্রোডাক্টিভিটি
Running Scripts
$chmod +x script.sh && ./script.sh
Make executable and run immediately.
এক্সিকিউটেবল করে সাথে সাথে চালায়।
$./script.sh &
Run script in background — terminal stays free.
ব্যাকগ্রাউন্ডে স্ক্রিপ্ট চালায় — টার্মিনাল মুক্ত থাকে।
$nohup ./script.sh > out.log 2>&1 &
Run in background, survive terminal close, save output to log.
ব্যাকগ্রাউন্ডে চালায়, টার্মিনাল বন্ধ হলেও চলে, লগে সেভ করে।
$echo $?
Exit code of last command. 0 = success, anything else = error.
শেষ কমান্ডের exit code। 0 = সফল, বাকি = ত্রুটি।
Bash Shortcuts
$Ctrl + C
Stop running command immediately.
চলমান কমান্ড তাৎক্ষণিক বন্ধ করে।
$Ctrl + R
Reverse search command history — type to find old commands.
কমান্ড হিস্ট্রিতে রিভার্স সার্চ করে।
$!!
Repeat last command. sudo !! = run last as root!
শেষ কমান্ড আবার চালায়। sudo !! = root হিসেবে চালায়!
$alias ll='ls -lah'
Create a shortcut command. Add to ~/.bashrc to make permanent.
শর্টকাট কমান্ড তৈরি করে। স্থায়ী করতে ~/.bashrc-এ যোগ করুন।
$history -c
Clear command history — anti-forensics 101!
কমান্ড হিস্ট্রি মুছে — অ্যান্টি-ফরেনসিক্স ১০১!

Advanced Commands ADVANCED

অ্যাডভান্সড কমান্ড
Pipe Chaining — পাইপ চেইনিং
$cat /etc/passwd | grep root
Output of cat piped into grep — find root entries.
cat-এর আউটপুট grep-এ পাঠায় — root এন্ট্রি খোঁজে।
$ps aux | grep apache | awk '{print $2}'
Find apache processes and extract just their PIDs.
Apache প্রসেস খুঁজে শুধু PID বের করে।
$find / -name "*.log" 2>/dev/null | xargs grep -l "password"
Find all log files containing "password". Pentesters love this!
সব লগ ফাইলে "password" খোঁজে। পেন্টেস্টারদের প্রিয়!
User & Sudo Enumeration
$sudo -l
What can THIS user sudo? Privilege escalation starting point #1.
এই ইউজার কী sudo করতে পারে? প্রিভিলেজ এস্কেলেশনের প্রথম পদক্ষেপ।
$cat /etc/passwd | cut -d: -f1
List all usernames on the system.
সিস্টেমের সব ইউজারনেম দেখায়।
$cat /etc/shadow
Password hashes — only readable as root. Gold if you can access it!
পাসওয়ার্ড হ্যাশ — শুধু root পড়তে পারে। যদি এক্সেস পাও তাহলে সোনার খনি!
$id && whoami && groups
Your current user, UID, and group memberships.
তোমার বর্তমান ইউজার, UID ও গ্রুপ সদস্যতা।
Cron Jobs — Persistence Vector
$crontab -l
List your scheduled cron jobs.
তোমার শিডিউল করা cron জব দেখায়।
$cat /etc/crontab
System-wide cron jobs — often run as root!
সিস্টেম-ওয়াইড cron জব — প্রায়ই root হিসেবে চলে!
$ls -la /etc/cron*
List all cron directories — look for writable scripts!
সব cron ডিরেক্টরি দেখায় — লেখার যোগ্য স্ক্রিপ্ট খোঁজো!
🚨
Writable cron script running as root = instant privilege escalation! / root হিসেবে চলা লেখার যোগ্য cron স্ক্রিপ্ট = তাৎক্ষণিক প্রিভিলেজ এস্কেলেশন!
Environment & PATH Hijacking
$echo $PATH
Show directories searched for commands. Add a writable dir first = PATH hijack!
কমান্ড খোঁজার ডিরেক্টরি দেখায়। প্রথমে লেখার যোগ্য dir যোগ করলে PATH হাইজ্যাক!
$export PATH=/tmp:$PATH
Add /tmp to the beginning of PATH — for PATH hijacking attacks.
/tmp কে PATH-এর শুরুতে যোগ করে — PATH হাইজ্যাক আক্রমণের জন্য।
$ls -ld $(echo $PATH | tr ':' '\n')
Check each directory in PATH — find writable ones!
PATH-এর প্রতিটি ডিরেক্টরি চেক করে — লেখার যোগ্য খোঁজো!

System Info & Kernel EXPERT

সিস্টেম তথ্য ও কার্নেল
Kernel & System Info
$uname -a
Kernel version + OS info — Google this for exploits!
কার্নেল ভার্সন + OS তথ্য — এটা দিয়ে এক্সপ্লয়িট গুগল করো!
$cat /proc/version
Detailed kernel version — useful for CVE matching.
বিস্তারিত কার্নেল ভার্সন — CVE ম্যাচিংয়ে কাজে লাগে।
$cat /etc/os-release
OS name and version details.
OS নাম ও ভার্সনের বিস্তারিত।
$free -h
Memory usage in human-readable format.
মেমোরি ব্যবহার সহজে বোধগম্য ফরম্যাটে।
$df -h
Disk space usage — see how full each partition is.
ডিস্ক স্পেস ব্যবহার — প্রতিটি পার্টিশন কতটা পূর্ণ।
Linux Capabilities
#getcap -r / 2>/dev/null
Find all binaries with Linux capabilities — overlooked priv esc vector!
Linux capabilities সহ সব বাইনারি খোঁজে — উপেক্ষিত প্রিভিলেজ এস্কেলেশন পথ!
#setcap cap_setuid+eip /usr/bin/python3
Set cap_setuid capability — can be used to get root shell!
cap_setuid capability সেট করে — root শেল পেতে ব্যবহার করা যায়!
🎯
cap_setuid or cap_net_bind_service = privilege escalation gold. Check GTFOBins! / cap_setuid বা cap_net_bind_service = প্রিভিলেজ এস্কেলেশনের সোনা!
Anti-Forensics
$echo > ~/.bash_history
Clear bash history — leave no trace of commands.
ব্যাশ হিস্ট্রি মুছে — কমান্ডের কোনো চিহ্ন রাখে না।
#cat /dev/null > /var/log/auth.log
Wipe authentication log — removes login evidence.
অথেন্টিকেশন লগ মুছে — লগইনের প্রমাণ সরিয়ে দেয়।
#rm -f /var/log/wtmp /var/log/btmp /var/log/lastlog
Delete login tracking files — wipes "last login" records.
লগইন ট্র্যাকিং ফাইল মুছে — "last login" রেকর্ড পরিষ্কার করে।
⚠️
Only use these on systems you OWN or have permission to test! / শুধুমাত্র নিজের বা অনুমতিপ্রাপ্ত সিস্টেমে ব্যবহার করুন!
SSH & Remote Access
$ssh user@ip
Connect to remote system securely.
নিরাপদে রিমোট সিস্টেমে সংযুক্ত হয়।
$ssh -L 8080:localhost:80 user@ip
SSH local port forwarding — tunnel traffic through SSH.
SSH লোকাল পোর্ট ফরওয়ার্ডিং — SSH দিয়ে ট্র্যাফিক টানেল করে।
$echo "ssh-rsa AAA..." >> ~/.ssh/authorized_keys
Add SSH key for passwordless login — persistence technique.
পাসওয়ার্ডহীন লগইনের জন্য SSH key যোগ করে — পার্সিস্টেন্স কৌশল।
$scp file user@host:/path/
Securely copy file to remote host.
রিমোট হোস্টে নিরাপদে ফাইল কপি করে।
